# Instruções para o Logo

Coloque o arquivo do logo da barbearia nesta pasta com o nome `logo.png`.

O logo deve ser:
- Formato: PNG com fundo transparente (preferível) ou JPG
- Tamanho recomendado: 500x500 pixels ou maior
- Boa qualidade para visualização em diferentes tamanhos

Após adicionar o logo, o site estará pronto para uso!

Você também pode adicionar outras imagens nesta pasta:
- favicon.ico (ícone do navegador)
- og-image.png (imagem para compartilhamento em redes sociais)
